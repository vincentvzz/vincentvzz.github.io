CSS
css transition sample code (make transition while hovering on a clickable link):

    a:link {
        color: white;
    }

    a:hover {
        text-decoration: underline;
    }

    a:visited {
        color: white;
    }
If you want to make a series of elements with 2 different colors:

    .job:nth-child(odd) {
        background-color: #7cd67f; /* background color for odd elements */
    }

    .job:nth-child(even) {
        background-color: rgb(184, 234, 179); /* background color for even elements */
    }
